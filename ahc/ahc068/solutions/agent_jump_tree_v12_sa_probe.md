# agent_jump_tree_v12_sa_probe

V11 ThickSafe を単調ガードとして残し、貪欲マクロ停止後だけに小さな焼きなまし継続を追加した検証候補です。

- 固定 seed、壁時計不使用
- 温度 1.5 → 0.05 の32段階
- 各段階は最大 `min(C, 512)` 提案・4受理、全体で最大16,384提案・128受理
- 代理評価は `prefix_len + 0.35 * Phi`
- SAの重い厳密補完は代理評価上位5チェックポイントだけ
- 従来V11 ThickSafeの全チェックポイント・72木プラン評価を先にそのまま実行
- 最終候補は従来V11を含む最短手数で選択し、リプレイ失敗時は検証済みV11へ戻す

これは探索余地を測るプローブであり、実行時間余裕を確認するまでは提出候補ではありません。この作成ターンでは入力ケースを実行していません。
